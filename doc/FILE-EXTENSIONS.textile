File Extensions

      <p> Here are the file extensions handled by !FunTester or its plug-ins:</p><p></p><table><tbody><tr><td><h3>Extension</h3></td><td><h3>Meaning/Content</h3></td></tr><tr><td>cfg</td><td>!FunTester's basic configurations.</td></tr><tr><td>dbt</td><td>Template for a database connection through a JDBC driver.</td></tr><tr><td>fat</td><td>[[AbstractTest|Abstract tests]], generated by !FunTester.</td></tr><tr><td>fp</td><td>[[Profile]] used by !FunTester for generating the [[AbstractTest|abstract tests]].</td></tr><tr><td>fs</td><td>Software data, created when we save a project in !FunTester.</td></tr><tr><td>ftr</td><td>Test results, generated by a plug-in.</td></tr><tr><td>fun</td><td>Software project, create by !FunTester.</td></tr><tr><td>fv</td><td>[[Vocabulary]] used by a software project to describe the use cases' steps.</td></tr><tr><td>pcfg</td><td>Plug-in configuration, for a software project.</td></tr><tr><td>plg</td><td>Plug-in descriptor.</td></tr></tbody></table>All these files are in JSON format.<p></p>
